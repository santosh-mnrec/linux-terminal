##Linux terminal using node.js and socket.io
###Demo Site
[Demo](https://nodeterm.herokuapp.com//)
###How to install
``` sh
  git clone https://github.com/santosh-mnrec/terminal.git
  cd DIR
  npm install
  node index.js
  ```
**open browser and type IP:PORT/index.html**
